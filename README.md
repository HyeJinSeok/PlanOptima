## 1. H2 데이터베이스 서버 실행
   
먼저 H2 데이터베이스를 TCP 서버 모드로 실행해야 합니다.<br>
터미널이나 커맨드 라인에서 다음 명령어를 실행합니다:<br>
java -jar h2*.jar -tcp -tcpAllowOthers -ifNotExists
